To-do List

Este projeto é uma aplicação de lista de tarefas (To-do List) desenvolvida com HTML, CSS e JavaScript puro, com foco em manipulação do DOM, eventos e persistência de dados no navegador.

O objetivo do projeto é permitir que o usuário gerencie suas tarefas diárias de forma simples, podendo adicionar, concluir, filtrar e remover tarefas, além de alternar entre modo claro e escuro.

Funcionalidades

Adicionar novas tarefas através do campo de texto pressionando Enter

Marcar e desmarcar tarefas como concluídas

Contador automático de tarefas ativas

Filtros de visualização:

Todas

Ativas

Concluídas

Remoção de todas as tarefas concluídas

Persistência dos dados utilizando localStorage

Alternância entre modo claro e modo escuro com salvamento da preferência

Interface responsiva
